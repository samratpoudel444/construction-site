# ICDC Website

## What is ICDC?

**ICDC** stands for **Indraakshi Divya Construction Development PVT. LTD.** It is a digital platform developed to establish the company's online presence and showcase its expertise in the construction sector.  

The website allows users to explore ICDC's completed projects, service offerings, team information, and company updates.

---

## Key Features

- **Project Showcase:** Displays completed projects and construction work performed by the company.  
- **Services Overview:** Showcases construction services and capabilities.  
- **Team Management:** Maintains staff records and team information.  
- **Company Updates:** Publishes company events and industry news.

---

## Tech Stack

- Frontend: React.js, Tailwind CSS, MUI  
- Backend: Node.js, Express.js , Typescript, redis 
- Database: PostgreSQL  
- Image Hosting: Cloudinary  
- API Management: Axios
